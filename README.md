# Finger Chooser

A simple touch-first web app that picks one random winner from 2+ fingers.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) on a mobile touch device.

## How it works

1. Place and hold at least 2 fingers on the screen.
2. A 3-second countdown starts automatically.
3. One finger is chosen at random.
4. Wait for cooldown, then play again.
